1. 使用现有用户 ID 8 创建临时服务地址。

6. Admin 查看订单详情，确认会员卡冻结数据可见。

9. 师傅端完成接单、出发、开始服务。
10. 师傅完成服务时提交实际服务 `60 分钟`。

12. 用户确认完成订单。
13. Admin 手动给该会员卡增加 30 分钟。
14. Admin 手动扣除 20 分钟。

17. 校验会员卡流水完整。
18. 清理所有本次测试数据。
